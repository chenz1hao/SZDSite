# SZDSite
某环保科技有限公司 官网

平台：.NET Framework 4.6.2

ASP.NET Core MVC

DataBase：SqlServer 2012

ORM采用EntityFramework 6

后台具有公司简介、企业文化、人才招聘的UEditor富文本编辑功能

同时有新闻发布、产品、解决方案的各类编辑管理，均可采用富文本编辑

自行定制前台样式后可用作各类企业官网

Model已经生成为Dll文件打包NuGet，自定义Model请自行制作Dll文件

目前已上线 http://www.szd-eco.com
